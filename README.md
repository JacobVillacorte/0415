# Anniversary Website

A sweet, bold one-page anniversary website.

## Customize

Edit config.js:

- partnerName
- relationshipStart (YYYY-MM-DD)
- anniversaryDay (MM-DD)
- heroMessage
- favoriteLine
- music.enabled
- music.url (for example: "./audio/our-song.mp3")
- music.volume (0 to 1)
- letterTitle
- letterBody
- timeline entries
- reasons list

## Run

Open index.html in your browser.
