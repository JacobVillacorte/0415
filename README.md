# Anniversary Website

A sweet, bold one-page anniversary website.

## Customize

Edit config.js:

- partnerName
- relationshipStart (YYYY-MM-DD)
- anniversaryDay (MM-DD)
- heroMessage
- favoriteLine
- music.enabled
- music.url (for example: "./audio/our-song.mp3")
- music.volume (0 to 1)
- letterTitle
- letterBody
- timeline entries
- reasons list

## Run

Open index.html in your browser.

## Publish (GitHub Pages)

Repo: https://github.com/JacobVillacorte/0415

1. Open your repo on GitHub.
2. Go to Settings > Pages.
3. Under Source, choose Deploy from a branch.
4. Select main branch and /(root), then Save.
5. Wait about 1 to 3 minutes.

Your live link will be:

https://jacobvillacorte.github.io/0415/
